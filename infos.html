Mehr über direkt-vom-hof.ch

Warum?

Diese Webapp entstand aus dem Wunsch, regionale Produkte sichtbarer zu machen und Produzent:innen direkt mit Konsument:innen zu verbinden – ohne Zwischenhandel, ohne Werbung, ohne Tracking.

Wie?

Die Anwendung ist bewusst einfach gehalten – sie basiert auf JavaScript, HTML und CSS. Gehostet wird sie auf GitHub Pages, und der GeoJSON-Datensatz liegt in einem öffentlichen GitHub-Repository. Die App lädt diesen Datensatz direkt über einen raw-Link. In regelmässigem Zeitabstand wird ein Query auf die Overpass API gemacht und falls es Änderungen gibt werden sie automatisch commited in das DB repository.

Was kann die App?

Du kannst landwirtschaftliche Betriebe in der Schweiz entdecken, die ihre Produkte direkt ab Hof verkaufen – ohne Zwischenhändler. Die Karte zeigt dir Standorte, Kontaktinformationen und ggf. Öffnungszeiten, basierend auf den OpenStreetMap-Daten.

Mitmachen?

Die Daten stammen aus OpenStreetMap. Neue Einträge oder Korrekturen kannst du direkt dort machen. Eine Anleitung findest du auf dieser Seite.
